嵌入式开发板老中青 SCM Arduino Galileo RaspberryPi

Arduino 主要是为非程序员、嵌入式工程师设计的硬件控制主控板。因为他的易用性和强大的可扩展能力，大量的设计师都会使用 Arduino 实现自己的项目。 Raspberry Pi 设计者本意是想为大学计算机系的新生提供一种简单的计算机，帮助他们去了解相对底层的计算机知识。因为他觉得现代的计算机已经太过复杂，抽象的层次太高，不利于底层知识的教学。所以……他就是一台迷你电脑。 而 Intel Galileo 比较复杂些，算是夹于 Arduino 和 Raspberry Pi 中间的东西吧，依赖于 Arduino 现有的资源（代码库和硬件模块），可以已有的 Arduino 用户很好的上手。另外，Intel Edison[5]（无比强大的产品~） 也是使用 Quark CPU 的哦~
