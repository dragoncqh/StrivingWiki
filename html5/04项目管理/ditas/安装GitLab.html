安装GitLab

操作系统为Ubuntu 16.04 LTS
  1. 安装配置依赖项:输入命令：sudo apt-get install curl openssh-server ca-certificates postfix
    (如果已经安装了Postfix来发送邮件请在安装期间选择 'Internet Site' . 你也可以用Sendmail或者 使用自定义的SMTP服务器来代替Postfix. 如果希望使用 Exim, 请 把它当做SMTP来配置.)
  2. 添加GitLab仓库,并安装到服务器上，可使用以下两种方式之一：
    1. 命令管道的安装方式：输入命令：curl -sS http://packages.gitlab.cc/install/gitlab-ce/script.deb.sh | sudo bashsudo apt-get install gitlab-ce
    2. 手动下载安装包(RPM/Deb)然后安装:输入命令：curl -LJO http://mirror.tuna.tsinghua.edu.cn/gitlab-ce/ubuntu/pool/trusty/main/g/gitlab-ce/gitlab-ce_XXX.debdpkg -i gitlab-ce-XXX.deb
      注：
      在浏览器中输入：http://mirror.tuna.tsinghua.edu.cn/gitlab-ce/ubuntu/pool/trusty/main/g/gitlab-ce就能看到要选择的gitlab版本软件 XXX表示版本号， 我用的是gitlab-ce_8.14.1-ce.1_amd64.deb
  3. 启动GitLab：输入命令：sudo gitlab-ctl reconfigure
    注：
    可以先编辑/etc/gitlab/gitlab.rb文件中的External Url为自已的域名，再启动
